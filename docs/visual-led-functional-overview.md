# Visual LED — Функциональные возможности (только визуализация)

Документ описывает функциональность инструмента визуализации экранов в браузере.  
В этом описании нет интеграций с БД, логированием и серверным хранением.

## 1. Работа со сценой и фоном

- Загрузка фонового изображения в рабочую область.
- Переключение между загруженными фонами.
- Автоматическая подстройка холста под размеры фонового изображения.
- Масштабирование и панорамирование рабочей области (zoom in/out/reset, перетаскивание вида).
- Ручная установка размеров viewport и кнопка быстрого вписывания содержимого в viewport.

## 2. Калибровка масштаба

- Режим задания масштаба по двум точкам на изображении.
- Ввод реальной длины в метрах между выбранными точками.
- Пересчет px↔м для всех дальнейших геометрических и размерных расчетов.

## 3. Создание и редактирование экранов

- Добавление экрана по 4 точкам (перспективный четырехугольник).
- Выбор активного экрана из списка.
- Перемещение экрана мышью.
- Редактирование формы экрана через перетаскивание углов.
- Переименование экранов.
- Удаление выбранного экрана (кнопкой и клавиатурой).
- Отображение геометрической информации по сторонам (AB/BC/CD/DA) и размеров в метрах.
- Ручная установка ширины/высоты экрана в метрах с перерасчетом геометрии.

## 4. Pixel pitch и кабинетная раскладка

- Выбор pixel pitch для каждого экрана (P1.9 / P2.6).
- Автозаполнение выбранного экрана кабинетами 0.5 × 0.5 м.
- Ручная корректировка количества кабинетов по колонкам и рядам.
- Очистка кабинетной раскладки.
- Подгонка размеров экрана под текущую кабинетную сетку.
- Включение/выключение отображения сетки кабинетов.
- Подсветка выхода кабинетов за границы экрана.
- Расчет и отображение количества кабинетов и итогового разрешения.

## 5. Многосценность

- Поддержка нескольких сцен в одном проекте визуализации.
- Создание новых сцен.
- Переименование сцен.
- Переключение между сценами через вкладки.
- Независимое состояние объектов для каждой сцены.

## 6. Assist (полуавтоматическая помощь)

- Запуск анализа сцены для поиска/предложения области экрана.
- Отображение направляющих и предложенного контура.
- Применение предложенного результата в активную сцену.
- Сброс/очистка результатов assist.
- Переключатель показа/скрытия assist-направляющих.

## 7. Видео и контент на экранах

- Загрузка видеофайлов в инструмент.
- Привязка видео к конкретному экрану.
- Просмотр визуального отображения видео в перспективе экрана.
- Очистка видео у выбранного экрана.
- Наличие встроенных demo-пресетов для быстрой проверки.

## 8. Отчеты и локальные артефакты

- Генерация HTML-отчета по активной сцене или по всем сценам.
- Открытие отчета в отдельной вкладке браузера.
- Копирование визуального состояния сцены в буфер обмена (с fallback-режимами).

## 9. UX и управление

- Поддержка drag-and-drop для загрузки файлов.
- Горячие клавиши для базовых действий редактирования.
- Визуальные статусы инструментов, подсказки и информационные панели.
- Отображение списка экранов и оперативной статистики текущей сцены.
